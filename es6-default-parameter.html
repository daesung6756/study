매개변수(parameter) 란?

매개변수(parameter)는 함수 실행시 인수를 함수 내부로 전달하기 위해 사용하는 변수를 뜻한다.

const testFunction = function ( 매개변수1, 매개변수2 ){
    console.log(매개변수1, 매개변수2);
    console.log(arguments[1])
    // 인자는 arguments 객체로도 접근이 가능하다.
}
// 함수 실행
testFunction(인자, 인자);

인자(argument) 란?

매개변수에 담겨 함수 내부에 전해지는 값을 인자라고하며,
함수를 호출할 때 arguments 객체도 내부로 전달된다.
arguments 객체는 함수를 호출할 때 넘긴 인자들이 배열로 저장되는 객체를 말한다.

기본 매개변수 정의란?

함수에는 정의 해놓은 매개변수가 있는데, 전달되는 인자값이 없을 경우 에러가 발생한다.
그래서 일반적으로 매개변수 정의가 되어있으면, 그 규칙을 잘 따르거나
함수 내부에 에러가 발생하지 않도록 코딩해야 한다.
이런 번거로움을 해결 할 수 있도록 es6에서는 기본값을 매개변수에 직접 정의 할 수 있게 되었다.

const testFunction = function ( parm1="첫번째", parm2=2, parm3='안녕하세요', parm4=10 ){
    console.log(parm1, parm2, parm3, parm4);
}
// 들어오는 인자 값을 확인해보면?
// 매개변수에 값이 담기지 않으면, 정의 해둔 기본값이 들어간다.
testFunction() // 첫번째,2, '안녕하세요', 10
testFunction(1) // 1,2, '안녕하세요', 10
testFunction(1,8) // 1,8, '안녕하세요', 10
testFunction(1,8, '반갑습니다.') // 1,8, '반갑습니다.', 10
testFunction(1,8, '반갑습니다.','홍길동') // 1,8, '반갑습니다.', '홍길동'
