Array.prototype

배열은 리스트와 비슷한 객체로 순회와 변형 작업을 수행하는 메서드를 갖게 된다.
배열의 내용은 언제든지 늘이거나 줄일 수 있다.
배열은 요소(value) index로 문자열을 사용할 수 없으며, 무조건 정수만 허용한다.

나의 생각 : 그래서 굳이 따지자면 배열도 객체라는 말을 한다는 것을 알게 되었다.

  • 누신기 ( accumulator ) : 콜백의 반환 값을 누적
  • 현재값 ( currentValue )
    • -
  • 현재 인덱스 (currentIndex )
  • reduce를 호출한 배열 ( array ) :
initialValue 값을 정의 하지 않으면, reduce()는 index 1부터 시작해 콜백 함수를 실행하고, 첫번째 index를 생략한다. initialValue을 정의하면 index 0에서 시작한다.
