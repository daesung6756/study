ECMAscript란 무엇인가?

ECMA 기관에서 만든 ECMA-262 표준을 따르는 스크립트다.
2015년도 버전을 줄여서 es6라고 한다.

ES6를 사용하는 이유

다른 버전에 비해 많은 시간과 많은 API 및 편의 기능을 제공하기 때문에 사용한다.

ES6를 사용 방법

외부의 JS파일을 따로 불러와서 사용하는 것이 아니라 ES6 문법을 그냥 사용 하면된다.
그러나 2022년 현재 아직 모든 브라우저에서 es6를 지원하지 않는다.
그래서 ie를 제외한 모든 브라우저가 지원하는 es5 버전으로 트랜스파일이 필요하다.
해당 트랜스파일을 하기 이해서는 Babel을 같이 사용 해야한다.
또한 es6에는 대체 할 수 없는 함수가 존재하기 때문에 이름 채워 줄 수 있는 polyfill을 같이 사용 해야한다.
Babel은 Webpack, Gulp, Grunt와 같은 자동화 빌드 도구을 사용하거나 CDN으로 불러와서 사용 할 수 있다.
