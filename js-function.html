자바스크립트 함수의 정의 및 종류

함수는 JavaScript에서 기본적인 구성 블록 중의 하나입니다.
함수는 작업을 수행하거나 값을 계산하는 문장 집합 같은 자바스크립트 절차입니다.
함수를 사용하려면 함수를 호출하고자 하는 범위 내에서 함수를 정의해야만 합니다.
[출처 : mdn web docs]

종류 정의 코드
일반 함수 (선언식)
  • - 이름이 있는 기명함수라고 한다.
  • - 호이스팅에 의해 최상의 scope로 이동 되기때문에 선언 전 후 사용이 가능하다.

    코드 작성

일반 함수 (표현식)
  • - 이름이 없는 익명함수라고 한다.
  • - 호이스팅에 의해 최상의 scope로 이동 하지만 변수 선언명만 이동 되기 때문에, 함수 호출이 불가능하다.

    코드 작성
}

중첩 함수 (Nested Function)
  • - 함수 내부에 정의되어 있는 함수를 말한다.
  • - 외부의 노출 시키지 않고, 내부의 함수를 사용함으로써 지역변수 scope으로 처리 해야 할때 사용한다.

    let n = 1;
    function secondFunc(){
        return n*10

    return secondFunc(); //10
}

콜백함수 (Callback Function)
  • - 특별한 선언법이나 문법이 따로 있는 것은 아니다. 단지 호출 방식에 의해 구분된다.
  • - 사용자가 명시적으로 사용하는 것이 아니라 정해 놓은 조건에 의해 호출 되는 함수를 말한다.

즉시 실행 함수 (immediate function)
  • - 사용자가 명시적으로 사용하는 것이 아니라 작성된 시점에 실행되는 함수이다.
  • - 익명함수 형태의 function 표현식에 ()괄호를 사용하여 처리한다.
  • - 끝에()를 추가하여 인자값을 넘길 수 있다.

2

(function(arg) {
    console.log('즉시 실행 함수 : ' + arg);
})('실험중'); //'즉시 실행 함수 실험중'
