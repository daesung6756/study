NodeJs란 무엇인가

Node.js는 구글 크롬의 V8 javascript 엔진으로 빌드 된 javascript 런타임이다.
Node.js를 통해 세계 최대의 오픈 소스 라이브러리르 보유하고 있는 npm 생태계에 접근을 할 수 있다.
Node.js를 설치하면 npm은 자동으로 설치되어 사용 가능하다.
Terminal, PowerShell, Command mode에서 사용이 가능하다.

자바스크립 런타임이란? : 특정 언어로 만든 프로그램을 실행할 수 있게 하는 환경을 말한다.
결국 자바스크립트 언어를 실행할 수 있는 실행기라 할 수 있다.

NPM 명령어 정리 NPM 사이트 바로가기

명령어 설명 기타
npm init package.json 파일 생성 -
npm help npm 전체 명령어에 대한 정보 확인. -
npm 명령어 help npm 특정 명령어에 대한 정보 확인. -
npm install npm 특정 모듈을 로컬에 설치. --save or -S : dependencies에 추가
--save-dev or -D : devDependencies에 추가
-g : global 패키지에 추가
npm i ┗ 축약 버전.
npm uninstall 설치한 모듈명 npm 특정 모듈을 로컬에서 삭제.
npm update 전체 모듈 버전 업데이트.

주의 : 모듈들의 버전 호환성 문제가 있어서 호환성을 확인하고 사용해야 한다.

 -
npm dedupe 중복된 모듈 정리 -
npm root node_modules 폴더의 위치를 알려준다. -
npm cache npm 내의 cache 목록 확인 -
npm cache clean cache 삭제 오류 발생시 자주 사용.
npm -version 버전 확인. -
npm -v ┗ 축약 버전. -
npm prune package.json 파일 내에 정의 되지 않은 모듈 삭제. -
npm start package.json의 scripts에 있는 start 정의했다면 명령어를 실행. scripts에 정의하지 않으면 node server.js가 실행
npm stop 실행 중인 npm을 중지. -
npm restart 실행 중인 npm을 stop 이후 start 실행. -
npm run package.json의 scripts에 정의 해서 사용. npm run 명령어
npm config npm 설정 조작 -
